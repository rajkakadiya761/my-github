# my-github
social media website with using MERN stack using login and singup , create and delete a post,  follow &amp;unfollow ,admin panel and live chat aplication
## Setup Instructions

### Backend
cd server
npm install
npm start

### Frontend
cd client
npm install
npm start

### Environment Variables
Create a `.env` file in server folder and add:

MONGO_URI=your_mongodb_uri
PORT=5000
